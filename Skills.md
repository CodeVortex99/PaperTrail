# Skills and Capabilities

Paper Trail is a comprehensive revision platform. Below are its core skills and technical capabilities.

## Core Features
- **Past Paper Library:** Dynamic path generation for A-Level and GCSE past papers across various exam boards (AQA, Edexcel, OCR).
- **Study Planner:** Weekly and monthly study timetable with persistence and task management.
- **Revision Lab:** SRS-based flashcards and interactive quiz lab for active recall.
- **Personalized Dashboard:** User profile management, exam countdowns, subject mastery tracking, and daily goals.
- **Subject-Specific Tools:** Checklist generator based on specifications and weakness trackers for GCSE/A-Level students.

## Technical Capabilities
- **Single Page Application (SPA):** Hash-based navigation with a custom lifecycle management system.
- **Performance Optimization:** Throttled event listeners, efficient 3D rendering (THREE.js), and optimized DOM updates.
- **Data Persistence:** Local-first storage using the `localStorage` API for privacy and offline access.
- **Responsive Design:** Mobile-first approach with a persistent hamburger menu and adaptive layouts.
- **Theming:** True black dark mode support using CSS variables.
- **Educational Tools:** UCAS tariff point calculator and grade boundary lookups.
