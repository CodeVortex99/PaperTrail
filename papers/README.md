# Past Papers — File Naming Convention

Drop your downloaded PDF files into the correct subfolder using this exact naming pattern:

```
papers/{Subject}/{Board}/{Session} {Type} - {Paper}.pdf
```

## Examples

| File path | What it is |
|---|---|
| `papers/Maths/AQA/June 2023 QP - Paper 1.pdf` | AQA Maths Paper 1, June 2023 Question Paper |
| `papers/Maths/AQA/June 2023 MS - Paper 1.pdf` | AQA Maths Paper 1, June 2023 Mark Scheme |
| `papers/Maths/Edexcel/June 2023 QP - Paper 31.pdf` | Edexcel Maths Paper 3 Statistics, June 2023 QP |
| `papers/Maths/Edexcel/June 2023 QP - Paper 32.pdf` | Edexcel Maths Paper 3 Mechanics, June 2023 QP |
| `papers/Physics/AQA/June 2022 QP - Paper 1.pdf` | AQA Physics Paper 1, June 2022 QP |
| `papers/Physics/AQA/June 2022 MS - Paper 1.pdf` | AQA Physics Paper 1, June 2022 Mark Scheme |
| `papers/Chemistry/Edexcel/June 2019 QP - Paper 2.pdf` | Edexcel Chemistry Paper 2, June 2019 QP |
| `papers/Biology/AQA/June 2018 QP - Paper 3.pdf` | AQA Biology Paper 3, June 2018 QP |

## Types
- `QP` = Question Paper
- `MS` = Mark Scheme

## Sessions
- `June 2023`
- `June 2022`
- `June 2019`
- `June 2018`

## Subjects & Folders
| Subject | Folder name |
|---|---|
| Maths | `Maths` |
| Further Maths | `Maths` (same folder) |
| Physics | `Physics` |
| Chemistry | `Chemistry` |
| Biology | `Biology` |

## Boards
- `AQA`
- `Edexcel`
